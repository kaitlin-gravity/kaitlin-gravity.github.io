Learning Resources

Tutorials

  • The Odin Project: An open-source full-stack curriculum. Their Foundations course offers guidance in HTML, CSS, and JavaScript.
  • Codecademy: Has free introductory tutorials on HTML and CSS but you should filter for free; all of their paid tutorials can be learned elsewhere.
  • W3Schools: Offers tutorials on everything website. A fantastic resource for looking up specific tag functions.
  • CSS Tricks: Guides to many different specific CSS uses, including plugins like SASS/SCSS/Bootstrap.
    • CSS Selectors
    • Media Queries for Standard Devices
    • Responsive Images

Tools

  • CodePen: A browser-based coding environment for practicing code and generating output without needing to create a website.

Web Hosting Services

A web hosting service is a service that offers the storage and capabilities required for you to create and maintain a site and make it accessible on the World Wide Web.

At minimum, modern webhosts support HTML, CSS, and JavaScript. Many also provide pre-built themes.

Free Web Hosts

  • GitHub Pages
  • Neocities
  • WordPress.com

Paid Web Hosts

Paid web hosting often supports more advanced apps and systems than simple code. They also offer domain name registration, which allows you to make the URL whatever you want. My personal website is hosted on Reclaim.

Glossary

World Wide Web (WWW): The "www" in most urls, the World Wide Web is an information system and the primary tool used for content sharing on the internet.

Open Source: Open source is source code that is made freely available for modification and redistribution. Open source models encourage collaboration and sharing of ideas. Open-source culture is intended to decentralize programming and development resources and make these skills available to everyone. Execution varies.

Hyper Text Markup Language (HTML): The basic markup language used to create websites. HTML defines the structure and content of website content and can include plain text, embedded video, embedded audio, and images.

Cascading Style Sheets (CSS): A style sheet language used to specify the presentation of a markup language, usually HTML. If it's pretty, it's using CSS.

JavaScript (JS): A programming language used to execute advanced web behavior like animation, response to events, and interactivity. With HTML and CSS, it provides the foundation for the World Wide Web.

What You See Is What You Get (WYSIWYG): Visual editors designed to convert rich text displayed in a User Interface into a code language. WYSIWYG editors for web development usually convert to HTML and CSS.
