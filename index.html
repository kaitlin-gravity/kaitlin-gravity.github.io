Web Hosting Services

A web hosting service is a service that offers the storage and capabilities required for you to create and maintain a site and make it accessible on the World Wide Web. There are free and paid webhosts which offer support for a variety of web technologies.

At minimum, modern webhosts support HTML, CSS, and JavaScript. Many also provide pre-built themes.

Glossary and Articles

World Wide Web

The "www" in most urls, the World Wide Web is an information system and the primary tool used for content sharing on the internet.

HTML/Hyper Text Markup Language

The basic markup language used to create websites. HTML defines the structure and content of website content and can include plain text, embedded video, embedded audio, and images.

CSS/Cascading Style Sheets

A style sheet language used to specify the presentation of a markup language, usually HTML. If it's pretty, it's using CSS.

JavaScript (JS)

A programming language used to execute advanced web behavior like animation, response to events, and interactivity. With HTML and CSS, it provides the foundation for the World Wide Web.

WYSIWYG/What You See Is What You Get

Visual editors designed to convert rich text displayed in a User Interface into a code language. WYSIWYG editors for web development usually convert to HTML and CSS.
